# Skills

A collection of AI agent skills for [Claude Code](https://claude.ai/code), installable via [skills.sh](https://skills.sh).

## Skills

### grill-me-product

A rigorous product thinking partner that interrogates a product idea, feature, or strategy until every assumption is surfaced and stress-tested. Draws on _The Mom Test_, _Shape Up_, _Thinking in Bets_, and _The Design of Design_.

### grill-me-architecture

A rigorous architecture reviewer that walks every branch of a design tree, resolves decision dependencies, and surfaces choices that will be expensive to change later. Draws on _A Philosophy of Software Design_, _How Buildings Learn_, _Systemantics_, and _The Design of Design_.

## Installation

Install both skills:

```sh
npx skills add sergdort/dot-files
```

Install individually:

```sh
npx skills add sergdort/dot-files --skill grill-me-product
npx skills add sergdort/dot-files --skill grill-me-architecture
```
